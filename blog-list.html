Note: Because this app is hosted on Heroku, it can take a few seconds to start. Once it is done 'spinning up' it should run smoothly.

This app is built entirely with Javascript. The backend is built with Node.js and Express while using Mongoose to communicate with MongoDB. Uses bcrypt, a hashing function to keep user's passwords safe. The frontend is built with React, Styled Components, and Axios for routing. Styled Components allows for CSS to be applied using Javascript.

This app taught me a great deal about communication between the various components of a website. It allowed me to practice working in development, testing, and production environments to quickly and efficiently develop.
