Help Desk

A support ticket app

Hover for video!

Code:
A single page support ticket application using Javascript, Node.js, and Express. The backend contains the functionality to send queries to a local MySQL database while following a RESTful architecture. The frontend is a UI to work with the current database entries. Each issue can be edited/closed with the changes updated on the database.

This application could benefit a team by allowing them to keep feedback in one place. They could place the 'New Issue' button on a user facing website. Then, the user could fill out a form with their issue and contact information which would be stored in a database for the team to address.
